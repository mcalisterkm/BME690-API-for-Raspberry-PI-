## Raspberry Pi version of the BME690 API

This is a port of the Bosch Sensortec BME690 air quality sensor for the Raspberry PI removing the COINS dependency and replacing with standard Raspbian/Linux functions. The original examples have been updated to add configuration by parameter files, and 64 bit PI binaries have been added. This allows you to change all of the key parameters without re-compiling. Each of the examples has a readme.md file explaining the configurable items.  

The original Bosch Sensortec BME690 API is available from the Bosch Sensortec GitHub. 

Note: This API is not backward compatible with BME688/680.
